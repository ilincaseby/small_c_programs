The maximum of three numbers_dynamic_alloc.c


This program shows you the maximum between 3 numbers given.
The thing that is different is the fact that I use structures for this.
Also they occupy the heap memory being allocated dynamic.
Commands:
1. add a b c
-Calculate the maximum-
2. free
-free the heap memory and exit the program-
